### next + eslint-config-next  16.2.3 → 16.2.6

Source: https://github.com/vercel/next.js/releases

#### v16.2.6

Security fixes and backported bug corrections.

High severity:
- Denial of Service with Server Components
- Middleware / Proxy bypass via segment-prefetch routes
- Connection exhaustion in Cache Components
- Dynamic route parameter injection
- WebSocket upgrade request forgery
- Pages Router i18n middleware bypass

Moderate severity:
- CSP nonce XSS vulnerabilities
- beforeInteractive script injection
- Image optimization denial-of-service
- RSC response cache poisoning

Low severity:
- Cache collision issues
- Middleware redirect cache poisoning

Bug fixes: HTTP access fallback preservation, route parameter handling, RSC request responses, header patching, deployment ID integration, URL encoding corrections.

#### v16.2.5

Same security categories as above (first patch to include them), plus additional backported fixes: HTTP access fallbacks, route parameter casing, RSC request handling, header patching, deployment ID caching, URL pathname encoding.

#### v16.2.4

Bug fixes only (no security):
- Turbopack filesystem watcher configuration fixes
- Google Fonts support for Windows on ARM64 (Reqwest dependency bump)
- Safari cache-buster scoping refinements
- Compiler support for boolean/number primitives in config definitions
- Turbo-tasks cell cleanup enhancements
- ChunkGroupInfo error messaging improvements
- ModuleBatchesGraph improvements
